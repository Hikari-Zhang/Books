# 战力与法宝设定表

本文档用于防止续写时战力膨胀和法宝归属混乱。

## 战力层级

| 层级 | 代表人物 | 写作约束 |
|---|---|---|
| 顶层命运级 | 邪皇渊乙、万劫真君、玄玄子、刑天、圣尊 | 不宜频繁正面下场；每次出手都应改变大局。 |
| 大罗/近大罗级 | 武翩跹、小妖后、元一太子、空空老仙、部分天庭上仙 | 小玄不能常规硬拼取胜，需借地势、法宝、计谋或多人合力。 |
| 太乙/宗主级 | 皇后、方少麟、李不、逍遥郎君、沈秀衣、千臂邪佛、邪军师等 | 可作为卷内主战力或博弈核心。 |
| 主角成长级 | 崔小玄、水若、雪妃、楚纯、李梦棠、夏小婉等 | 小玄强在多法门叠加与临场破局，其余人物应有明确专长。 |
| 军阵/群魔级 | 邪宗群魔、七绝界将领、天相宗机关军、皇朝军 | 用于大战场推进，不应随意被单人秒尽。 |

|---|---|---|

| 北溟玄数 | 已至第四境“抱拙”。 | 可用于洞察轨迹和延展刹那，不宜代替所有推理。 |

| 归墟本诀 | 已与雪妃共修，有第二重天阶基础。 | 可辅助疗伤、身法、海界线，但不能喧宾夺主。 |
| 役妖令 | 可召十三罪妖，摩珈女王第279章决定战局。 | 召唤次数、符石损耗、罪妖反噬需统一，摩珈不可频繁救场。 |

## 法宝使用规则

- 每场大战最多突出两到三件核心法宝，避免堆砌。
- 新法宝优先服务旧伏笔，不为单章爽点随意添加。
- 终局法宝体系建议围绕“先天太玄、圣器三元、不周钥、七邪覆、神骨剑”展开。
- 法宝破损、易主、献祭应写入章后状态记录，方便后续追踪。
