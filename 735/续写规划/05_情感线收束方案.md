# 情感线收束方案

本文档定义第280-479章情感线的优先级和边界。目标是保留原书多情风格，同时避免后期失焦。

## 总原则

- 水若是第一情感主线，必须在前期完成信任重建。
- 武翩跹是神话与深情并重的高权重线，应在中后段公开化。
- 皇后是政治、危险、激情与共治线，不应简单降格为后宫争宠。
- 雪妃、婀妍、楚纯、飞萝等都要有独立功能线，而非只服务吃醋。
- 第400章后不再新增主要女性情感对象。

## 主线情感

| 情感线 | 当前状态 | 收束方式 |
|---|---|---|
| 小玄与水若 | 水若得知小玄假冒天子及雪妃之事，出走后在279章重逢。 | 280-310完成摊牌。水若可以生气、质问、离席，但不再长期失踪。终局为第一伴侣。 |
| 小玄与武翩跹 | 常羊秘境后情感深重，武翩跹离开迷楼。 | 400章前后公开摆脱纯师徒名分，刑天线结束后确认彼此位置。 |
| 小玄与皇后 | 假夫妻、政治盟友、真实情欲交织。 | 皇后承认小玄不是棋子而是共同执棋者，终局辅政与情分并存。 |

## 重要副线

| 情感线 | 收束建议 |
|---|---|
| 小玄与雪妃 | 通过水若摊牌转为姐妹共盟，雪妃不再做误会核心。 |
| 小玄与楚纯 | 从被救创伤和“硬要跟着”开始，定为知己或侧室线，不抢水若主位。 |
| 小玄与婀妍 | 以建木心髓、虚照境、复国线回归，终局可作为政治盟约与深情线。 |
| 小玄与飞萝 | 回归时处理小妖后托付与旧情，不再制造主三角。 |
| 小玄与李梦棠 | 回收杨奕嫉妒和竹林情债，给李梦棠尊严与承诺。 |
| 小玄与糖妃/汤妃 | 配合方少麟“斩妖妃”条件，不能只作为艳情角色，应有政治与魔线价值。 |

## 时间轴

| 章段 | 情感任务 |
|---|---|
| 280-299 | 水若、楚纯、雪妃、皇后同台前奏；小玄先坦白假天子和雪妃，暂时隐瞒武翩跹造成后续雷点。 |
| 300-339 | 水若与雪妃摊牌，皇后定规则；小玄学会不再用插科打诨逃避责任。 |
| 340-379 | 邪宗与妖界大战中，楚纯、婀妍、李梦棠等副线取得阶段归宿。 |
| 380-419 | 武翩跹回归，水若面对小玄更深的旧情和救父线。 |
| 420-459 | 终局大战以“不离不弃”为情感兑现，不再用新误会推动剧情。 |
| 460-479 | 劫后安置所有重要情感关系，明确谁同行、谁辅政、谁守一方。 |

## 禁忌

- 不再让水若连续多次误会出走。
- 不让元一太子因受伤或身世获得情感洗白。
- 不用女性角色牺牲来廉价推动小玄突破，除非前文已充分铺垫且不可替代。
- 不把皇后写成突然嫉妒失智。
- 不让雪妃永远夹在水若和小玄之间反复受苦。
