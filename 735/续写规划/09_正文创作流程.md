# 正文创作流程

本文档用于从第280章开始逐章写作，并在每章后维护状态。

## 单章流程

1. 读取对应细纲，确认本章功能。
2. 列3到5个场景，不超过本章承载能力。
3. 写正文时保留原书风格：古典仙侠、危机反转、轻佻对白、权谋暗流、情感暧昧、章末悬念。
4. 写完后更新 `章后状态记录.md`。
5. 每5章做一次连续性检查。
6. 每20章做一次卷末复盘，再决定是否调整下一卷细纲。

## 单章写作卡模板

```markdown
# 第X章：标题

## 本章功能

- 推进主线：
- 情感变化：
- 伏笔回收：
- 新增钩子：

## 场景顺序

1. 场景一：
2. 场景二：
3. 场景三：

## 必须出现

- 人物：
- 法宝/设定：
- 章末钩子：

## 禁止写法

- 不新增无关大设定。
- 不跳过上一章遗留危机。
- 不让战力突破无代价发生。
```

## 每5章检查

每写完5章，检查以下内容：

- 人物是否符合 `03_核心人物弧光表.md`。
- 法宝归属是否符合 `02_战力与法宝设定表.md`。
- 是否至少推进一个旧伏笔。
- 是否新增了未记录的新伏笔。
- 情感线是否重复同一种误会。
- 小玄是否仍保持顽劣、机智、担当并存。

## 每20章复盘

每卷结束后，复盘以下内容：

- 本卷主冲突是否闭环。
- 卷初承诺的伏笔是否回收。
- 下一卷钩子是否明确。
- 哪些人物需要暂时退场。
- 哪些设定需要补入人物卡或设定表。
- 是否需要微调十卷总纲。

## 第280章写作卡

### 本章功能

- 推进主线：接续第279章鲲鹏发狂，完成战后混乱开场。
- 情感变化：楚纯观察小玄优先照顾水若，水若开始重新看见小玄的担当。
- 伏笔回收：邪宗众魔称少主、观音甘露、鲲鹏失控。
- 新增钩子：鲲鹏悲鸣与鹏腹秘藏。

### 场景顺序

1. 鲲鹏冲天，殿宇翻覆，众魔自救。
2. 小玄以“御”字稳住岛礁和断殿，救下水若、楚纯、百宝娘娘附近众人。
3. 千臂邪佛和邪军师请命，率众魔锁鲲鹏足脉，小玄第一次正式下少主令。
4. 小玄把观音甘露先递给水若，楚纯看在眼中，水若既气又心软。
5. 鲲鹏再度悲鸣，众人发现它并非只受惊，似有体内秘物牵动。

### 必须出现

- 人物：小玄、水若、楚纯、百宝娘娘、千臂邪佛、邪军师。
- 法宝/设定：御字神通、观音甘露、役妖令余威、鲲鹏。
- 章末钩子：鲲鹏悲鸣，邪军师察觉鹏腹有圣器气息。

### 禁止写法

- 不让小玄轻松完全制服鲲鹏。
- 不让水若立刻原谅小玄。
- 不让楚纯创伤在本章完全消失。
